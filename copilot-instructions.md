# Copilot Instructions

- **Primary rule:** Always write source code, identifiers (variable, function, class names), and comments in English.
- For user-facing text that requires localization, keep the code and comments in English and provide translations via the project's localization mechanisms.
- This rule applies to generated code, suggestions, and any inline documentation produced by Copilot.

- **Note:** This project intentionally targets .NET 6 because it is the last .NET version supported on Windows 7. Do not change the `TargetFramework` without considering Windows 7 compatibility.
